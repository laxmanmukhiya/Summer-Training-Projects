1. Image Decolorization / Color Channel Manipulation
A Python-based image processing project that demonstrates RGB color channel manipulation using NumPy and Pillow.
The program adjusts the intensity of the red channel with clipping to produce controlled color enhancement and visualizes the output using Matplotlib.

2. Negative Image Converter
A simple image processing application built with Python that converts a normal image into its photographic negative by inverting RGB pixel values.
Developed using NumPy, Pillow, and Matplotlib to demonstrate fundamental digital image processing concepts.
